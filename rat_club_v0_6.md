## 1. 背景

2026 年初台北市大安區出現 25 年來首例漢他病毒死亡案例。後續多處公共場所（雙連市場、木柵動物園、信義商圈、台北車站）頻繁目擊鼠群白天活動。

### 主要成因（已識別）
- **都更驅趕效應：** 大安區大量拆除老舊建物，原棲息鼠群被迫遷移
- **食源管理失敗：** 市場收攤後廚餘暴露、民眾餵食流浪動物散落飼料
- **氣候因子：** 全球暖化延長鼠類繁殖活躍季節
- **下水道連通性：** 老舊管線破損提供地下遷移通道

### 政治脈絡（干擾因子）
年底縣市長選戰升溫，鼠患議題被政治化。KMT 陣營指控公民通報為「造謠」，ratdar.taipei 開發者 Wildcat Yang 已對相關立委提告。通報資料同時受兩股力影響：關注度偏差灌水 + 政治壓力消音。

## 2. 核心方法：改裝版 Auerbach Capture-Recapture

### 原始方法（NYC, 2014）
統計學家 Jonathan Auerbach 用 NYC 311 通報資料，將 capture-recapture 改裝為：

```
T = S1 × S2 ÷ R

T  = 估計有鼠地段總數
S1 = 第一時間窗口被通報的地段數
S2 = 第二時間窗口被通報的地段數
R  = 兩個窗口都被通報的地段數（recapture）
```

**NYC 結果：** 45,000 有鼠地段（4.75% of all lots）× 50 隻/地段 ≈ 200 萬隻

### 台北適用條件
- **資料來源：** ratdar.taipei API（v0.4 起直連）
- **地段定義：** 500m grid cells（0.005°），可切換 200m（0.002°）或 1.1km（0.01°）
- **時間窗口：** 依資料日期自動對半切分
- **假設前提：**
  - 所有有鼠地段被通報的機率相等（已知不成立，但先用）
  - 兩個時間窗口之間總有鼠地段數穩定（短期內合理）

### 已知偏差（五層模型，v0.6 更新）

| 層級 | 偏差來源 | 方向 | 發現者 |
|------|----------|------|--------|
| ① 平台基礎建設 | 審核機制轉換（AI→人工），審核 AI 為 Claude | S1 灌水 | Mellisa |
| ② 觀察者密度 | 觀光區/商業區 vs 住宅區通報密度不均 | 熱門區高估 | — |
| ③ 人口年齡結構 | 年輕人通報、老社區沉默 | 老城區低估 | — |
| ④ 制度性通報障礙 | 學校、醫院、軍營、五院——「不能滑手機」的地方 | 機構用地低估 | Mellisa |
| ⑤ 觀察者駐留性 | RR% 混合量測鼠穩定性與觀察者穩定性，商業區觀察者不重複導致 RR 結構性偏低 | 商業/觀光區系統性低估 | Mellisa + Reflex |

**第④層案例：** 景美某學校四周被通報熱點包圍，但校內通報為零（學生上課不能用手機）。學校可能是未偵測的 colony 核心，周圍通報是溢出邊緣。此偏差非隨機分佈——學校通常位於社區中心。

## 3. 棲地分層估算模型

白天目擊數 → 乘以棲地對應 multiplier → 估算區域族群量

### 3.1 老舊社區（屋齡 30 年以上住宅區）

| 參數 | 值 | 備註 |
|------|-----|------|
| 遮蔽物密度 | 高 | 管道間、天花板夾層、鐵皮加蓋、牆壁裂縫 |
| 食源可及性 | 中 | 家庭廚餘，受垃圾不落地政策壓制 |
| 下水道狀態 | 差 | 老舊管線破損率高，連通性強 |
| 結構入侵點 | 多 | 管線孔洞未封 |
| 白天目擊 multiplier | **1:30–50** | 遮蔽物多，大部分不可見 |
| 族群特徵 | 穩定駐留型 | 世代連續，低遷移率 |

**鐵皮加蓋案例：** 通報者描述「大量勞贖在老公寓鐵皮通過，聲音劇烈」。鐵皮加蓋形成天花板上方的遮蔽通道，連通整排住戶。地面投藥無法觸及此類族群；且鼠若在天花板內中毒死亡，將導致腐敗、蛆蟲等二次問題。

### 3.2 醫院社區

| 參數 | 值 | 備註 |
|------|-----|------|
| 遮蔽物密度 | 中 | 建物較新但管線通道多 |
| 食源可及性 | 高且穩定 | 廚房、餐廳、廚餘量大 |
| 衛生管理 | 有制度 | 執行力看醫院等級 |
| 白天目擊 multiplier | **1:20–30** | 高人流壓抑白天活動 |
| 族群特徵 | 高繁殖率 + 定期壓制 | 食源穩定推升繁殖，消毒壓制存活 |

**台大醫院案例：** 通報描述「有大老鼠，很愛吃東西，常常會在美食街吃東西」。「常常」= 反覆目擊 = confirmed resident。語氣缺乏恐慌 = 通報者已習慣其存在。醫院屬第④層偏差（制度性黑洞），此通報可能來自訪客。

### 3.3 市場週邊

| 參數 | 值 | 備註 |
|------|-----|------|
| 遮蔽物密度 | 中 | |
| 食源可及性 | **極高，有時間週期** | 收攤→清運之間 2-4 hr 開放食源窗口 |
| 廚餘暴露時間 | 關鍵變因 | 直接影響 carrying capacity |
| 白天目擊 multiplier | **1:15–25** | 食源豐富，白天活動比例較高 |
| 族群特徵 | 密度最高、波動最大 | 與市場營運週期連動 |

### 3.4 都更工地週邊（本次特殊因子）

| 參數 | 值 | 備註 |
|------|-----|------|
| 遮蔽物密度 | 變動中 | 拆除過程遮蔽物快速消失 |
| 食源可及性 | 低（原棲地）→ 向外擴散找食 | |
| 遷移壓力 | **極高** | 本次安鼠之亂的主要驅動因子 |
| 白天目擊 multiplier | **不適用** | 這類是遷移中個體，不是穩定族群 |
| 族群特徵 | 擴散型 | 被擠出的鼠往周圍社區和下水道遷移 |

### 3.5 捷運站與地下基礎設施（新增 v0.5）

| 參數 | 值 | 備註 |
|------|-----|------|
| 遮蔽物密度 | 高 | 隧道、管道、電纜井 |
| 食源可及性 | 中-高 | 站內商店、地下街、旅客棄食 |
| 溫度 | 恆溫 | 冬暖夏涼 |

**大安捷運站案例：** 500m 半徑內 18 筆通報、RR 40%，但 RR 低於周邊公園（60%）。判讀：捷運站是覓食中繼站而非巢穴，鼠來此覓食後沿走廊返回定居點。

### 繁殖力快算
最大理論值（不受環境限制）：
```
1 對鼠 × 8 仔/胎 × 6 胎/年 = 48 仔/年
子代 12 週成熟 → 年內可再繁殖
理論年增長：一對 → 數百（指數增長）
實際受 carrying capacity 壓制，野外存活率 ~15%
```

### 5.7 RR30 Rule — Recapture Rate 觀察基準（新增 v0.6）

> **核心發現：** RR% 同時量測兩件事——**鼠的駐留穩定性**與**觀察者的駐留穩定性**。

住宅區兩者皆高：同一批居民走同一條路，看到同一批鼠，反覆通報同一個 grid cell。商業/觀光區兩者皆低：不同的過路人看到鼠，通報一次，不會再來。RR% 與其說量「鼠有多少」，不如說量「該地的人鼠關係有多穩定」。

**RR30 閾值定義：**
- **RR ≥ 30%：** 即使在觀察者稀釋和棲地異質性的條件下，仍有至少一個穩定的 colony 核心在撐。列入 confirmed 監測。
- **RR < 30%：** 不代表無鼠，但代表沒有穩定到能被通報系統反覆捕捉的駐留族群。可能為注意力外溢、過路通報、或觀察者密度不足。

**RR30 驗證案例：大安區三層異質性**

大安區整體 RR 39%，由三種完全不同的都市組織混合產出：

| 都市組織 | 觀察者穩定性 | 鼠駐留穩定性 | 對 RR 的貢獻 |
|----------|-------------|-------------|-------------|
| 老公寓社區（屋齡 30+年） | 高——居民固定 | 高——世代駐留 | ⬆ 拉高 |
| 大安森林公園 | 低——散步者路線不固定 | 高——大量遮蔽 + carrying capacity | → 中等 |
| 新建豪宅 | 極低——空屋率高 | 低——管線密封 | ⬇ 稀釋 |

老社區的真實訊號強到足以扛住其他兩塊的稀釋，區級仍站在 39%。而 per-sentinel 數據顯示 colony 核心（仁愛敦南圓環 60%、敦安公園 60%）被區級平均壓了近一半。

**諷刺結構：** 新建案施工期間的都更驅趕效應將周圍老社區鼠群擠入公園和地下管道。豪宅本身管線密封、鼠進不去，但「被蓋出來」這個過程加劇了周邊的鼠患。直接受益者同時是間接加害者，且因不居住而不會通報——成為觀察盲區。

### 5.8 行政區都市紋理分析（新增 v0.6）

> **核心洞察：** 公衛分析通常以行政區為單位計算，但行政區邊界是政治產物而非生態邊界。理解每個區的都市紋理（urban fabric）才能正確判讀 RR% 的意義。

#### 中正區 —— 制度性黑洞的極致案例

| 紋理要素 | 特徵 | 對鼠群的意義 |
|----------|------|-------------|
| 五院（行政院、立法院、司法院、考試院、監察院） | 高度管制區域，門禁+警衛+不可滑手機 | 第④層偏差：制度性通報障礙最極端的形式 |
| 學校（北一女、建中、師大附中等） | 學生上課不能用手機 | 與景美案例相同的觀察黑洞 |
| 植物園 + 綠廊道 | 大面積遮蔽、水源、低人為干擾 | 理想棲地——carrying capacity 高 |
| 南門市場 | 食源核心 | 已列為二級 sentinel |
| 台大醫院 | 食源穩定（美食街）+ 第④層偏差 | 已有 confirmed resident 通報 |
| 博物館群（國立歷史博物館、郵政博物館等） | 大型建物地下室、老建築管線 | 遮蔽物密度高 |
| **實際居民密度** | **全台北最低之一** | 觀察者 pool 極度稀薄 |

**判讀：** 中正區的 RR 21% 不應按照 RR30 Rule 判定為「非穩定族群」。這個區的觀察者結構極度特殊：

1. **白天觀察者**是通勤公務員——每天走固定路線（捷運出口→辦公室），但身份每天輪替，且工作時間不會通報
2. **夜間觀察者**幾乎為零——五院下班後清空，學校放學後清空，真正住在這裡的居民極少
3. **鼠的夜間活動不可能低**——植物園+南門市場+老建築管線提供完整的棲地三要素（食源、遮蔽、水源），夜間無人干擾反而是最佳活動時段

中正區的 21% 應理解為：**在觀察者幾乎全面缺席的條件下，僅靠白天通勤族的零星通報仍能產出的訊號。** 如果將觀察者密度標準化，中正區的「修正 RR」可能不低於萬華。

**中正區應列為特殊 sentinel 區——不適用 RR30 Rule 的標準判讀，需獨立框架。**

### 5.9 行為觀察摘要（通報文字質性分析）

#### 恐懼反應崩潰
多個地點獨立通報相同行為特徵：
- 永春公園：「老鼠不怕人也不怕狗」
- 北車 K7：「不怕人大米奇」
- 晴光公園：「坐在椅子上他會靠過來」

**判讀：** 多點同時出現恐懼閾值下降 = 族群層級行為，非個體差異。密度高到個體競爭壓力超過人類迴避本能。

#### 死亡同類迴避失效
萬華通報：「死翹翹在路口，活的在旁邊跑步」

**判讀：** 鼠對死亡同類有先天迴避反應（演化防禦機制）。此行為消失代表：密度過高無替代路徑、死亡頻繁導致 habituation、或食源價值壓過迴避驅力。此訊號比「族群數量多」更嚴重——是行為結構層級的瓦解。

#### 垂直擴散
中正區通報：「老鼠入侵 10 樓住家，咬破紗窗進屋咬食物」

**判讀：** 褐家鼠一般活動於地面層。10 樓入侵代表使用建築內部管道間/電纜井垂直移動，或為偏好高處的黑家鼠（Rattus rattus / roof rat）。主動破壞紗窗 = 地面食源競爭激烈到驅動垂直擴散。

#### 萬華擴散走廊
通報者自行描繪路徑：「果菜市場→富民路→一直延伸至青年公園，數量相當龐大」

**判讀：** 食源核心（市場 3.3）→ 線性街道擴散 → 大型綠地（公園 3.1）的遷移走廊。青年公園游泳池兒童遊戲區已發現排泄物——公衛紅線。

#### 食源共享
通報：「小公園有人在餵鴿子，五月九日早上發現有一窩老鼠」

**判讀：** 鴿子飼料散落 → 鼠的食源。餵食行為同時 amplify 鼠患和流浪動物二次中毒風險。

## 6. 預測假說（2026-06-07 驗證）

### H1：投藥無效假說
> 台北市府以大規模投藥為主要策略，未處理食源管理。一個月後：
> - 鼠類目擊通報短期下降（毒死一批 + 恐懼迴避）
> - 但 6 月進入夏季繁殖高峰，剩餘族群快速補回
> - 淨效果：6 月中目擊數回到或超過 5 月水準
>
> **v0.5 補充：** 投藥在老舊公寓（鐵皮加蓋類型）為雙輸策略——不吃藥鼠在天花板跑，吃了藥鼠在天花板腐爛。政治端收到的訊號永遠為負。

### H2：毒藥食物鏈效應 → 毒藥依賴迴圈（擴充 v0.5）
> 大量投放抗凝血鼠藥 → 中毒鼠行動遲緩被猛禽/貓/流浪犬捕食 → 二次中毒 → 天敵死亡 → 生物防治鏈斷裂 → 鼠群反彈更快 → 需要更多投藥
>
> **結構性陷阱：** 每一輪迴圈都強化對化學防治的依賴，同時削弱生態系統的自我調節能力。一旦踏入就難以退出——退路（天敵）已被殺光。
>
> **餵食行為的雙重 amplification：** 流浪動物餵食點的殘留飼料 = 鼠的食源（驗證 H5），同時將流浪犬貓固定在毒餌區域 → 二次中毒射程內。
>
> - 預測：6 月開始出現鳳頭蒼鷹或其他猛禽異常死亡報導
> - 已知基線：過去四年超過百隻死亡猛禽中 61% 驗出鼠藥殘留
>
> **生物防治替代方案備註：** 紐約 R.A.T.S. 組織使用梗犬類（Rat Terrier, Jack Russell）進行夜間獵鼠，可繞過 H2 毒藥迴圈。臺北推行可能面臨動保爭議。

### H3：擴散假說
> 台北都更驅趕 + 投藥驅趕 → 鼠群向外圍遷移
> - 預測：新北市（特別是與台北交界的區域）通報上升
> - 已有訊號：新北已出現漢他病毒確診案例
> - **v0.5 更新：** 10 樓入侵案例顯示垂直擴散路徑啟動

### H4：政治週期假說（擴充 v0.5）
> 選舉炒作 → 通報量與實際鼠密度脫鉤
> - 預測：選舉新聞降溫後，通報量下降，但實際鼠密度未變
> - **v0.5 更新：** 政治力不只拉高通報（關注度），也試圖壓制通報（指控造謠）。兩股力同時作用於同一份資料——灌水和消音並存。KMT 攻擊公民通報可信度 → Wildcat 反手提告，將消音策略的法律成本拉高。

### H5：食源才是真正的 rate-limiting factor
> 如果市府在一個月內未改善市場廚餘管理、未推動密閉式垃圾桶：
> - carrying capacity 不變 → 族群會自動回復到投藥前水準
> - 紐約經驗：密閉垃圾桶政策後目擊數下降 14-30%

### H6：通報量衰減假說（已部分驗證 ✓）
> **v0.5 驗證結果：**
> - 半衰期確認為 ~1.5 天（5/4: 329 → 5/6: 105）
> - 5/13 降至 15 筆（peak 的 4.5%），接近基線
> - **Sentinel 分離確認：** 萬華衰減曲線平坦（5/10 反彈至 8），信義崩至 2，士林崩至 0 → 全域下降確認為注意力偏差消退，非實際改善
> - 基線估計：每日 15-25 筆

### H7：可見性偏差干預假說
> 市府可能採取「移除地表遮蔽物」策略（砍草叢、修剪植被）來回應通報熱點
> - 預測：信義等商業區開始砍除景觀植被
> - 實際效果：鼠群轉入地下/管道間/大樓夾層，通報數下降但族群密度不變
> - 驗證：如果信義通報下降但 sentinel sites 維持 → 假性改善

## 附錄 B：公式速查

```
【Auerbach Capture-Recapture】
T = S1 × S2 ÷ R
T = 有鼠地段估計總數
S1 = 第一期通報地段數
S2 = 第二期通報地段數
R = 兩期皆通報地段數

【Per-Sentinel 半徑 RR%（v0.5 新增）】
給定 sentinel 座標 (lat₀, lon₀) 和半徑 r：
1. 篩選 haversine(lat₀, lon₀, latᵢ, lonᵢ) ≤ r 的所有鼠蹤通報
2. 將篩選後的通報依 grid 解析度歸入 cells
3. 依日期對半切 S1/S2，算 R 和 RR%
優勢：同區不同微環境（市場 vs 辦公區）產出不同 RR%

【RR30 Rule（v0.6 新增）】
RR% ≥ 30% → Confirmed：至少一個穩定 colony 核心
RR% < 30% → 非穩定駐留族群（或觀察者密度不足）
例外：制度性觀察黑洞（中正區類型）需獨立評估
RR% 同時量測：鼠駐留穩定性 × 觀察者駐留穩定性

【區域族群估算】
P = Σ (Oi × Mi)
P = 區域族群估計
Oi = 第 i 棲地類型的白天目擊數
Mi = 第 i 棲地類型的 multiplier

【繁殖增長（無限制）】
Nt = N0 × e^(rt)
r = intrinsic rate of increase（鼠 ≈ 0.03-0.05/day）
實際受 K (carrying capacity) 壓制 → logistic growth
```
